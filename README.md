# CropTool
Python script to minimize blank in a matlab or python plot.

![CropTool](https://i.imgur.com/OgiaoHb.png)

## How to use ?
+ Download the script (Crop.py)
+ Windows user : Launch anaconda prompt
+ Go to the path of the script :

```
cd username/mypath/...
```
+ Execute the script :

```
python Crop.py -h
```
